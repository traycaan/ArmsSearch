Using the Search

  • Use quotes to search exact strings (e.g. "my unique example")
  • Using "site: " will only search that site (e.g. site:mtsite.com)
  • The tabs at the top of the search results are sites that have information on a wide range of firearms, using them will search that site
  • Put ".." between numbers to search a range (e.g. Walther Model 3..9)
  • Using "OR" will search for articles with either term (e.g. Walther OR Colt)
  • Using "AND" will search for articles with both terms (e.g. Walther AND Colt)
  • Using "filetype: " will search for items of that type (e.g. filetype:pdf)
  • Using a "-" before a word excludes it from the search (e.g. -Colt  1911)
  • Using "-site: " excludes a site from the search (e.g. -site:myste.com)
